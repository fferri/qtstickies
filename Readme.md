# QTStickies

A cross platform, modern take on OSX's stickies.

Status: Very much WIP

What works:
  * Draggable
  * Collapsable
  * Editable
  * Resizeable
  * Yellow

TODO for parity:

## Buidling

```
$ qmake
$ make
```

## Running

`$ open qtstickies.app`
